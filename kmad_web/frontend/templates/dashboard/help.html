Help
Sequence

The sequence should be either in FASTA format or a plain protein sequence, in case of a single sequence input. In case of multiple sequences input the accpeted format is FASTA. If the chosen action is 'refine' the sequences have to be aligned (FASTA).

Action

In the action field you specify which action do you want to perform: either 'predict', 'predict and align', 'align', or 'refine'. The 'predict' option will use the programs chosen by the user to predict disorder. 'Align' will either just perform a multiple sequence alignment if multiple sequences are provided, or if the input is a single sequence it will run a BLAST search against SwissProt and align the first 75 homologues with e-value lower or equal 1e-05. 'Predict and align' will simply perform both of the mentioned actions, and refine will run a KMAD refinement on an alignment provided by the user.

Gap penalties

gap opening penalty
- penalty for opening a new gap
gap extension penalty
- penalty for extending an already existing gap
end gap penalty
- gap penalty for a gap et the end (or beginning) of a sequence


Feature weights (ωft)

motif score
score for aligning two motifs of the same type
PTM score
score for aligning two PTMs of the same type together
domain score
score for aligning two domains of the same type and at the same time penalty for aligning two domains of a different type


User defined features

You can add new features to your sequences by specifiying their exact positions in the sequences or by pattern - the feature will be assigned to every position where it matches the provided regex.

Results

'Submit' button submits the job and redirects you to the output page. You can download both the disorder prediction and the alignment (in FASTA format).
